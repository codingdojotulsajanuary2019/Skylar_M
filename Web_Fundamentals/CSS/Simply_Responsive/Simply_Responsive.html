@media queries

CSS uses the @media rule to detect information aboout the device being used to access the website. For example, it can be used to detect the size of the device, it's capabilitiies, and its orientation. For this course, we will be using it to detect size, but keep in mind that it is capable of much much more.

Desktop: Width greater than or equal to 1024px

Tablet: Width between 481px and 1023px

Phone: Width less that or equal to 480px
